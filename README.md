# MediaPipe Computer Vision Projects 🎥

A collection of real-time computer vision projects built using Python, OpenCV, and MediaPipe. This repository demonstrates face tracking, hand landmark detection, gesture recognition, and finger counting applications.

---

## 🚀 Features

- Real-time face landmark detection
- Face mesh visualization
- Hand landmark tracking
- Finger counting system
- Gesture recognition using hand landmarks
- Webcam-based live processing

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

---

## 📂 Project Structure

```text
mediapipe-projects/
│
├── face_landmarks.ipynb
│   └── Detects and visualizes facial landmark points in real-time
│
├── face_mesh.ipynb
│   └── Generates a detailed facial mesh using MediaPipe Face Mesh
│
├── hand_landmark_detection.ipynb
│   └── Detects and tracks hand landmarks in real-time
│
├── finger_counter.py
│   └── Counts the number of fingers shown using hand landmark detection
│
├── gesture.py
│   └── Recognizes hand gestures:
│       • 👍 Thumbs Up
│       • ✌️ Victory Sign
│
├── requirements.txt
│   └── Required project dependencies
│
└── README.md
```
